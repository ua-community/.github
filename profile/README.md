<h2>Our Projects</h2>

<p>
  Projects built by community members that align with our goals and are open to users and contributors.
</p>

<h3>xe / darc</h3>

<p>
  <a href="https://xe.dev">https://xe.dev</a>
</p>

<ul>
  <li>
    <strong>Xenon</strong>: A framework for building secure, powerful applications with browser-like capabilities
    using web standards.
  </li>
  <li>
    <strong>Darc</strong>: A maximalist agentic browser built on Xenon, focused on user agency, modularity,
    and reasonability.
  </li>
</ul>

<h3>Tiles</h3>

<p>
  <a href="https://www.tiles.run">https://www.tiles.run</a>
</p>

<ul>
  <li>
    <strong>Tiles App</strong>: A private AI assistant with offline, user-owned memory.
  </li>
  <li>
    <strong>Tilekit SDK</strong>: A Modelfile-based Rust SDK that powers model deployment and execution across
    the Tiles ecosystem.
  </li>
</ul>

<h3>Peek</h3>

<p>
  <a href="https://github.com/autonome/peek">https://github.com/autonome/peek</a>
</p>

<ul>
  <li>
    An experimental user agent that deconstructs the browser to let you work with the web directly.
    More window manager than application.
  </li>
</ul>

<h3>Diffuse</h3>

<p>
  <a href="https://elements.diffuse.sh/">https://elements.diffuse.sh</a>
</p>

<ul>
  <li>
    A collection of components and software for listening to audio from multiple sources across devices
    and the web, designed to support a flexible and personal listening experience.
  </li>
</ul>
